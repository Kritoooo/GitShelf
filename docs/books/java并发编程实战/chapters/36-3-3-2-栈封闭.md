# 3.3.2 栈封闭

栈封闭是线程封闭的⼀种特例，在栈封闭中，只能通过局部变量才能访问对象。正如封装能使得代码更容易维持不变性条件那样，同步变量也能使对象更易于封闭在线程中。局部变量的固有属性之⼀就是封闭在执⾏线程中。它们位于执⾏线程的栈中，其他线程⽆法访问这个栈。栈封闭（也被称为线程内部使⽤或者线程局部使⽤，不要与核⼼类库中的ThreadLocal混淆）⽐Ad-hoc线程封闭更易于维护，也更加健壮。

对于基本类型的局部变量，例如程序清单3-9中loadTheArk⽅法的numPairs，⽆论如何都不会破坏栈封闭性。由于任何⽅法都⽆法获得对基本类型的引⽤，因此Java语⾔的这种语义就确保了基本类型的局部变量始终封闭在线程内。
