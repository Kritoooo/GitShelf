# 程序清单3-2 非线程安全的可变整数类

```java
@NotThreadSafe   
public class OnClickListener { private int value; public int get() { return value; } public void set(int value) { this.value = value; } 
```

![](../images/ebe79f084bedb72d4a66bac116f7c2d17622e35e9263928c6604820864100f74.jpg)

在程序清单3-3的SynchronizedInteger中，通过对get和set等方法进行同步，可以使MutableInteger成为一个线程安全的类。仅对set方法进行同步是不够的，调用get的线程仍然会看见失效值。
