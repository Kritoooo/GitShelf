# 5.5 同步⼯具类

在容器类中，阻塞队列是⼀种独特的类：它们不仅能作为保存对象的容器，还能协调⽣产者和消费者等线程之间的控制流，因为take和put等⽅法将阻塞，直到队列达到期望的状态（队列既⾮空，也⾮满）。

同步⼯具类可以是任何⼀个对象，只要它根据其⾃⾝的状态来协调线程的控制流。阻塞队列可以作为同步⼯具类，其他类型的同步⼯具类还包括信号量（Semaphore）、栅栏（Barrier）以及闭锁（Latch）。

在平台类库中还包含其他⼀些同步⼯具类的类，如果这些类还⽆法满⾜需要，那么可以按照第14章中给出的机制来创建⾃⼰的同步⼯具类。

所有的同步⼯具类都包含⼀些特定的结构化属性：它们封装了⼀些状态，这些状态将决定执⾏同步⼯具类的线程是继续执⾏还是等待，此外还提供了⼀些⽅法对状态进⾏操作，以及另⼀些⽅法⽤于⾼效地等待同步⼯具类进⼊到预期状态。
