# 第3章 对象的共享

第2章的开头曾指出，要编写正确的并发程序，关键问题在于：在访问共享的可变状态时需要进⾏正确的管理。第2章介绍了如何通过同步来避免多个线程在同⼀时刻访问相同的数据，⽽本章将介绍如何共享和发布对象，从⽽使它们能够安全地由多个线程同时访问。这两章合在⼀起，就形成了构建线程安全类以及通过java.util.concurrent类库来构建并发应⽤程序的重要基础。
