# 4.3.2 独⽴的状态变量

到⽬前为⽌，这些委托⽰例都仅仅委托给了单个线程安全的状态变量。我们还可以将线程安全性委托给多个状态变量，只要这些变量是彼此独⽴的，即组合⽽成的类并不会在其包含的多个状态变量上增加任何不变性条件。

程序清单4-9中的VisualComponent是⼀个图形组件，允许客户程序注册监控⿏标和键盘等事件的监听器。它为每种类型的事件都备有⼀个已注册监听器列表，因此当某个事件发⽣时，就会调⽤相应的监听器。然⽽，在⿏标事件监听器与键盘事件监听器之间不存在任何关联，⼆者是彼此独⽴的，因此VisualComponent可以将其线程安全性委托给这两个线程安全的监听器列表。
